이전

색종이

NYPC 2018 · 본선

N×MN\times MN×M 크기의 격자 모양의 스케치북과 다양한 크기의 색종이들을 가지고 있다. 위에서 부터 iii번째 행, 왼쪽에서 부터 jjj번째 열에 해당하는 격자는 (i,j)(i, j)(i,j)번 격자라고 부른다. 예를 들어, 가장 왼쪽 위의 격자는 (1,1)(1, 1)(1,1), 오른쪽 아래의 격자는 (N,M)(N, M)(N,M) 번 격자다.

당신에겐 크기와 색을 마음대로 정할 수 있는 색종이가 NMNMNM개 주어진다. 색종이의 색은 111이상 333이하의 정수로 표현된다.

이 스케치북에 격자에 맞는 색종이들을 스케치북에 잘 붙여서 원하는 그림의 모양을 만들고자 한다. 각 격자의 색상은 마지막에 붙인 색종이의 색상이 된다. 단, 각 격자에 붙어있는 색종이가 반드시 있도록 붙여야한다.

색종이들을 적게 사용해서 원하는 그림을 만드는 방법을 구하여라.

입력 형식

첫째 줄에는 세로의 길이 NNN, 가로의 길이 MMM, 사용할 수 있는 색종이의 개수 VVV가 공백으로 구분되어 주어진다.

다음 NNN개의 줄에는 111이상 CCC이하의 정수 MMM개가 공백으로 구분되어 주어진다. iii번째 줄의 jjj번째 수는 원하는 그림의 (i,j)(i, j)(i,j)번 격자의 색을 의미한다.

CCC는 별도로 입력으로 주어지지 않는다. CCC에 관련해서는 아래 입력 데이터 및 생성 방법을 참고하자.

출력 형식

첫째 줄에 사용한 색종이의 개수 TTT를 출력한다.

다음 TTT개의 줄에는 색종이를 붙인 순서대로 색종이에 대한 정보를 줄로 구분하여 출력한다. 색종이의 정보는 xxx, yyy, zzz, www, ccc를 차례로 공백으로 구분하여 출력한다. 이는 사용한 색종이의 왼쪽 위 격자가 (x,y)(x, y)(x,y), 오른쪽 아래 격자가 (z,w)(z, w)(z,w), 색이 ccc라는 것을 의미한다. (1≤x≤z≤N1 \le x \le z \le N1≤x≤z≤N, 1≤y≤w≤M1 \le y \le w \le M1≤y≤w≤M, 1≤c≤31 \le c \le 31≤c≤3 를 만족하게 출력해야 한다.)

555, 666번 데이터를 제외하고는, 각 칸을 111이상 CCC이하의 수를 랜덤으로 정하여 생성되었다. 555, 666번 데이터는 첫번째 직사각형으로 전체를 덮은 후, 직사각형 V−1V-1V−1개를 랜덤하게 생성하여 덮는 방법으로 생성되었다.

VVV는, 111, 222, 333, 444번 데이터의 경우에는 최소의 가능한 색종이 개수 +1+1+1로, 555번, 666번의 경우 데이터를 만들 때 사용한 직사각형의 갯수로, 777번, 888번의 경우 NM(C−1)C\displaystyle\frac{NM(C-1)}{C}CNM(C−1)​로, 999번, 101010번의 경우에는 NM(C−1)2C\displaystyle\frac{NM(C-1)}{2C}2CNM(C−1)​로 결정되었다.

배점

배점은 각 테스트 데이터당 101010점이다.

T>NMT > NMT>NM 혹은, 색종이를 붙였을 때 그림이 일치하지 않으면, 000점을 받는다.

T<VT < VT<V이면 101010점, V≤T≤NMV \le T \le NMV≤T≤NM 이면, T=VT = VT=V일때 101010점, T=NMT = NMT=NM 일때 111점을 기준으로 점수가 선형으로 주어진다. 즉, 점수는

1+9×NM−TNM−V1 + 9 \times \frac{NM-T}{NM-V}1+9×NM−VNM−T​

이다. 그 후, 각 테스트케이스의 점수는 0.050.050.05 단위로 버림된다.
