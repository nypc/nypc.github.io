이전

물 뿌리기

NYPC 2025 · 본선

NNN개의 구역이 길이가 111인 N−1N - 1N−1개의 도로로 연결되어 있다. 각 구역에는 111 이상 NNN 이하인 서로 다른 정수로 번호가 매겨져 있고, 어떤 두 구역을 고르더라도 도로를 통해서 연결되어 있다. 이제 여러분은 총 MMM번의 물 뿌리기 작업을 하려고 한다.

한 번의 물 뿌리기 작업은 (t,u,v,ru,rv)(t, u, v, r_u, r_v)(t,u,v,ru​,rv​) 555개의 값으로 정의할 수 있다. 이는 시각 ttt에, uuu번 구역으로부터 rur_uru​개 이하의 도로를 통해 갈 수 있으면서 동시에 vvv번 구역으로부터 rvr_vrv​개 이하의 도로를 통해 갈 수 있는 모든 구역에 물을 뿌린다는 뜻이다. 엄밀히 말하면, 시각 ttt에 uuu번 구역으로부터 거리가 rur_uru​ 이하인 구역들의 집합과 vvv번 구역으로부터 거리가 rvr_vrv​ 이하인 구역들의 집합의 교집합에 해당하는 구역들에 물을 뿌린다.

또한, 물이 뿌려진 구역에서는 KKK시간 후 인접한 구역으로 물이 전파되며, 전파된 구역 역시 동일하게 KKK시간마다 인접한 구역으로 물을 전파한다.

여러분은 각 구역에 대해 물이 뿌려지거나 전파되는 최초의 시각을 구해야 한다.

N=4N = 4N=4, M=1M = 1M=1, K=1K = 1K=1이고 구역과 도로가 아래의 그림과 같은 예제를 생각해보자.

  • 먼저 한 번의 물 뿌리기 작업이 (1,1,4,2,2)(1, 1, 4, 2, 2)(1,1,4,2,2)이라고 하자.
    • 시각 t=1t = 1t=1 때, 111번 구역에서 거리가 222 이내인 동시에 444번 구역에서 거리가 222 이내인, 222번과 333번 구역에 물이 뿌려진다.
    • K=1K = 1K=1시간 후인 시각 t=2t=2t=2 때, 222번 구역 인접한 111번 구역에, 333번 구역과 인접한 444번 구역에 물이 뿌려진다.
  • 그렇지 않고, 물 뿌리기 작업이 (1,1,4,1,1)(1, 1, 4, 1, 1)(1,1,4,1,1)이라고 하자.
    • 시각 t=1t = 1t=1 때, 111번 구역에서 거리가 111 이내인 동시에 444번 구역에서 거리가 111 이내인 구역은 없기 때문에, 물이 뿌려지는 구역은 없다.
    • 물이 뿌려진 구역이 없기 때문에 전파되는 구역도 없다.

입력 형식

첫 줄에 구역의 수 NNN, 물뿌리기 작업의 수 MMM, 물이 전파되는데 걸리는 시간을 나타내는 정수 KKK가 공백으로 구분되어 주어진다. (2≤N≤100 000;2 \le N \le 100\,000;2≤N≤100000; 1≤M≤200 000;1 \le M \le 200\,000;1≤M≤200000; 1≤K≤501 \le K \le 501≤K≤50)

그다음 줄부터 N−1N - 1N−1개의 줄에 걸쳐, 각 줄에 각 간선이 연결하는 구역의 번호를 나타내는 두 정수 UiU_iUi​, ViV_iVi​가 공백으로 구분되어 주어진다. (1≤Ui,Vi≤N;1 \le U_i, V_i \le N;1≤Ui​,Vi​≤N; Ui≠ViU_i \ne V_iUi​=Vi​)

그다음 줄부터 MMM개의 줄에 걸쳐, 각 줄에 각 물 뿌리기 작업을 나타내는 다섯 개의 정수 ttt, uuu, vvv, rur_uru​, rvr_vrv​가 공백으로 구분되어 차례대로 주어진다. (1≤t≤5 000 000;1 \le t \le 5\,000\,000;1≤t≤5000000; 1≤u,v≤N;1 \le u, v \le N;1≤u,v≤N; 0≤ru,rv≤N0 \le r_u, r_v \le N0≤ru​,rv​≤N)

예제 설명

입력 예제 1에서, 111번 방과 222번 방을 연결하는 복도 위에 몬스터가 있는 방을 하나 추가하면 문제의 조건을 만족한다.
