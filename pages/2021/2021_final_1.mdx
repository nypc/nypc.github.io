7-세그먼트 표시 장치는 7개의 LED로 된 획을 이용하여 정보를 표시하는 장치이다. 이를 이용하면 0부터 9까지 숫자를 다음과 같이 표시할 수 있다.

<Figure
  src="/2021/7segment01-7a87b622.png"
  caption="&lt;그림 1&gt; 7-세그먼트 표시 장치의 각 숫자 표시 방법"
  width={540}
/>

위 그림을 보면, 0, 1, 2, 5, 6, 8, 9는 7-세그먼트 표시 장치를 180도 회전해서 읽어도 자기 자신, 또는 다른 숫자를 표시하는 것을 알 수 있다. 회전했을 때 1이 약간 달라지지만 1이라는 걸 알아볼 수 있다. 회전해서 읽은 수는 각각 0, 1, 2, 5, 9, 8, 6이 된다.

각 자릿수가 0, 1, 2, 5, 6, 8, 9 중 하나이고 0으로 시작하지 않는 양의 정수 $a$가 있다고 하자. $a$를 7-세그먼트 표시 장치로 표시한 다음, 이 장치를 180도 회전해서 읽은 수를 $b$라고 하자. 두 수의 차 $a-b$를 $x$라고 하자. 예를 들어, $a=65$이면 $b=59$이고, $x=a-b=6$이다.

반대로 수 $x$가 주어졌을 때, 위 조건을 만족하며 $a-b = x$가 되는 두 수 $a$, $b$를 구하고 싶다. 만약 이런 $a$, $b$가 여러 가지 존재한다면, 가장 작은 $a$ 값을 구하려고 한다. 이 값을 구해주는 프로그램을 작성하시오.

## 입력 형식

첫 줄에 테스트케이스의 수 $T$가 주어진다. $(1 \le T \le 100)$

각 테스트케이스는 한 줄로 이루어지며,
각 줄에 하나의 정수 $x_i$가 주어진다.
$(1 \le x_i \le 30\,000)$

$a-b=x_i$가 되는 $a$가 존재하지 않는 경우는 입력으로 주어지지 않는다.

## 출력 형식

$T$ 개의 줄에 걸쳐 답을 출력한다.
$i$ 번째 줄에는 $a-b=x_i$가 되는 가장 작은 $a$ 값을 출력한다.

## 해설

<Solution>
  사용할 수 있는 수는 $0$, $1$, $2$, $5$, $6$, $8$, $9$의 일곱 가지다. 그리고,
  답이 되는 수의 자릿수가 $9$자리 이하라는 것도 알 수 있다. 이를 종합하여,
  가능한 모든 수에 대해 차를 미리 계산하는 전처리를 통해 모든 답을 미리 구해놓을
  수 있다. 테스트케이스마다 답을 구하지 않고, 모든 답을 미리 구해 놓은 뒤,
  테스트케이스마다 입력을 받고 미리 구해 놓은 답만 출력한다면 실행 시간을
  최적화할 수 있다.
</Solution>
