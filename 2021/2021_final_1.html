7-세그먼트 표시 장치

7-세그먼트 표시 장치는 7개의 LED로 된 획을 이용하여 정보를 표시하는 장치이다. 이를 이용하면 0부터 9까지 숫자를 다음과 같이 표시할 수 있다.

위 그림을 보면, 0, 1, 2, 5, 6, 8, 9는 7-세그먼트 표시 장치를 180도 회전해서 읽어도 자기 자신, 또는 다른 숫자를 표시하는 것을 알 수 있다. 회전했을 때 1이 약간 달라지지만 1이라는 걸 알아볼 수 있다. 회전해서 읽은 수는 각각 0, 1, 2, 5, 9, 8, 6이 된다.

각 자릿수가 0, 1, 2, 5, 6, 8, 9 중 하나이고 0으로 시작하지 않는 양의 정수 aaa가 있다고 하자. aaa를 7-세그먼트 표시 장치로 표시한 다음, 이 장치를 180도 회전해서 읽은 수를 bbb라고 하자. 두 수의 차 a−ba-ba−b를 xxx라고 하자. 예를 들어, a=65a=65a=65이면 b=59b=59b=59이고, x=a−b=6x=a-b=6x=a−b=6이다.

반대로 수 xxx가 주어졌을 때, 위 조건을 만족하며 a−b=xa-b = xa−b=x가 되는 두 수 aaa, bbb를 구하고 싶다. 만약 이런 aaa, bbb가 여러 가지 존재한다면, 가장 작은 aaa 값을 구하려고 한다. 이 값을 구해주는 프로그램을 작성하시오.

입력 형식

첫 줄에 테스트케이스의 수 TTT가 주어진다. (1≤T≤100)(1 \le T \le 100)(1≤T≤100)

각 테스트케이스는 한 줄로 이루어지며, 각 줄에 하나의 정수 xix_ixi​가 주어진다. (1≤xi≤30 000)(1 \le x_i \le 30\ 000)(1≤xi​≤30 000)

a−b=xia-b=x_ia−b=xi​가 되는 aaa가 존재하지 않는 경우는 입력으로 주어지지 않는다.

출력 형식

TTT 개의 줄에 걸쳐 답을 출력한다. iii 번째 줄에는 a−b=xia-b=x_ia−b=xi​가 되는 가장 작은 aaa 값을 출력한다.

입력 예제

2
3
6

출력 예제

9
65

채점 방식

입력 케이스들은 다음과 같은 종류로 구별되며, 한 종류의 케이스를 다 맞혀야 그 종류에 배정된 점수를 받을 수 있다.

  • 종류 1 (31점): xi≤100x_i \le 100xi​≤100
  • 종류 2 (32점): xi≤3 000x_i \le 3\ 000xi​≤3 000
  • 종류 3 (37점): 추가적인 제한 조건이 없음.
